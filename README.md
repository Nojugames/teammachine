### Coding style
Only use Wordpress coding style

#### Setup in PHPStorm
1. File->Settings
2. Editor->Code Style->PHP
3. Change Scheme from Defalt to Project
4. On the right, click "Set from...", choose WordPress
5. Click "Apply"
